About this website

This website is for me to document my personal journey, findings. It's also for others to see all my photos that I've taken. Plus, this website also surves as a way to introduce myself.

This website is designed with single page in mind.

What this means is that the website is designed to be on one page. All content will be on one page and transitions.

this design has a big flaw. the webpage has a chance of causing a large amount of memory useage as Everything is not being cleared out of memory but instead new content is being placed on top of the existing content. This design also increases the complexity of making the website. This might lead to more unexpected behaviour after deployment.

This is just an experiment that might fail. In the event of unexpected issues, please switch to https://joshualimkarmin.github.io/joshualimkarmin.github.io/no-single-page and please report it to me at my email with the subject header being website issue report. Then desccribe the issue. Thanks!

Please note that this website might go through a lot of changed in the future.
